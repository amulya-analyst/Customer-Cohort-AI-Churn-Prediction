# Customer Cohort Analysis & AI Churn Prediction

## Author
Amulya

## Project Overview
Customer cohort analysis extended with AI-powered churn prediction 
model to identify at-risk customers and reduce churn.

## Tools Used
- Python (pandas, scikit-learn, matplotlib)
- SQL
- Power BI
- Excel

## AI Model
- Algorithm: Random Forest Classifier
- Accuracy: 78%
- Dataset: 594 customers, 3000 transactions
- Key Finding: Total Revenue is the top churn driver

## Key Insights
- 141 out of 594 customers identified as churned
- Active customers generate significantly higher revenue
- AI model predicts at-risk customers with 78% accuracy

## Files
- cohort_analysis.ipynb — Full analysis + AI model
- churn_predictions.csv — AI predictions for all customers
- churn_feature_importance.png — Feature importance chart
